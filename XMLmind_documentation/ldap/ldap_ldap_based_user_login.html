9. LDAP based user login

The task was to enable our SDI8b machine to handle users by querying our LDAP Server on our SDI8a machine. To achieve this we installed the package libpam-ldapd and provided the necessary data during the configuration process. The we told the system to use the LDAP data by changing the following lines in /etc/nsswitch.conf:

passwd:         files ldap
group:          files ldap
shadow:         files ldap

Next we shut down the nscd deamon to prevent usage of invalid cached data. We restarted our VM and then we tested the LDAP setup by executing:

id bean

The result showed that we were successful:

uid=1(daemon) gid=1(daemon) Gruppen=1(daemon)
