8. Extending an existing entry

To add the objectClass posixAccount and its according attributes uidNumber, gidNumber and homeDirectory we created a LDIF file with:

dn: uid=bean,ou=devel,ou=software,ou=departments,dc=betrayer,dc=com
changetype: modify
add: homeDirectory
homeDirectory: /home/bean
-
add: uidNumber
uidNumber: 1
-
add: objectClass
objectClass: posixAccount
-
add: gidNumber
gidNumber: 1
-

Then we applied it with:

ldapmodify -x -D "cn=admin,dc=betrayer,dc=com" -w sdi8a -H ldap:// -f ~/modify_bean.ldif
