9. Finishing the installation

To finish the installation, we restarted our Apache server with:

systemctl restart apache2

Now the Nextcloud service was set up and we could login under sdi8b.mi.hdm-stuttgart.de/nextcloud which forwarded to https://sdi8b.mi.hdm-stuttgart.de/nextcloud/index.php/login.
