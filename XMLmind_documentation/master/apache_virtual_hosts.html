2. Virtual Hosts

We created virtual hosts for every path extension which includes /manual, /nv012...

Each virtual host was created in a separate configuration file.

In order to change the domain for each configuration file we changed the ServerName and the ServerAlias of the file.

Next, we used the Alias keyword to redirect the domain to its corresponding directory.

Example manual:

<VirtualHost *:80>
    ServerName manual.mi.hdm-stuttgart.de
    ServerAlias www.manual.mi.hdm-stuttgart.de
    ServerAdmin webmaster@localhost
    DocumentRoot /var/www/manual.mi.hdm-stuttgart.de
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
        Alias "/" "/usr/share/doc/apache2-doc/manual/"
</VirtualHost>
<Directory /var/www/manual.mi.hdm-stuttgart.de>
    Options Indexes FollowSymLinks
    AllowOverride All
    Require all granted
</Directory>

We added the subdomains to our server's DNS entries and needed to add them to each of our computers' local hosts in order to access the newly created subdomains.
