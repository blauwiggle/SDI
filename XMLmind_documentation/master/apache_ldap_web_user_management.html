6. Providing WEB based user management to our LDAP Server

We installed the ldap-account-manager:

apt install ldap-account-manager

We updated the configuration under /var/lib/ldap-account-manager/config/lam.conf

admins: cn=admin,dc=betrayer,dc=com
treesuffix: dc=betrayer,dc=com

We were able to log in our web interface under http://sdi8b.mi.hdm-stuttgart.de/lam using our LDAP credentials admin and sdi8b.
