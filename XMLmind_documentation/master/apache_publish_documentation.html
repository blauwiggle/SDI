7. Publish your documentation

We uploaded our documentation to /home/sdidoc and set up an alias for the URL path /doc to this location by adding these lines to our site configuration files:

Alias "/doc" "/home/sdidoc"

<Directory /home/sdidoc>
    Require all granted
</Directory>

The documentation is now available under https://sdi8b.mi.hdm-stuttgart.de/doc.
