1. Prerequisites

The task was to install a file cloud system using Nextcloud. To do this, we followed the guide at https://help.nextcloud.com/t/complete-nc-installation-on-debian-9-stretch-and-manual-update/21881 . The first thing was to install all necessary packages: 

apt install vim unzip sudo
apt install apache2 mariadb-server libapache2-mod-php
apt install php-gd php-json php-mysql php-curl
apt install php-intl php-mcrypt php-imagick
apt install php-zip php-xmlwriter php-xmlreader php-xml php-mbstring php-simplexml
