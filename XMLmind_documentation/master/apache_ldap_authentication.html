4. LDAP authentication

The goal here was to set up a simple LDAP authentication for our web server's manual directory. T achieve this, we first enabled the necessary modules with a2enmod ldap and a2enmod authnz_ldap.

After creating a corresponding user on our LDAP server running on our A machine, we added the following lines to /etc/apache2/conf-available/apache2-doc.conf:

AuthName "Manual"
	AuthBasicProvider ldap
	AuthType Basic
	AuthLDAPURL "ldap://sdi8a.mi.hdm-stuttgart.de/dc=betrayer,dc=com"

After restarting the server, you are now required to provide user credentials to access this directory under sdi8b.mi.hdm-stuttgart.de/manual.
