6. Nextcloud installation and configuration

First, we executed the Nextcloud installation script with:

sudo -u www-data php occ maintenance:install --database "mysql" --database-name "nextcloud" --database-user "nextcloud" --database-pass "sdi8bnextclouddbpassword" --admin-user "ncadmin" --admin-pass "sdi8bnextcloudadminpassword"

Then we added our server to the trusted domains by adjusting the configuration file under /var/www/nextcloud/config/config.php:

'trusted_domains' => 
  array (
    0 => 'localhost',
    1 => 'sdi8b.mi.hdm-stuttgart.de',
  )
