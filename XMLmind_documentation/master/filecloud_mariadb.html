5. Configuring MariaDB

As with our Apache server, we used the existing MariaDB database from the previous tasks. So we just added a database and a user for our Nextcloud system by executing the following SQL satement:

mysql -u root -p
CREATE DATABASE nextcloud;
GRANT ALL ON nextcloud.* to 'nextcloud'@'localhost' IDENTIFIED BY 'sdi8bnextclouddbpassword';
FLUSH PRIVILEGES;
exit
