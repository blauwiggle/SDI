11. Difficulties

There were two major difficulties while doing this task:

  • The tutorial on the Nextcloud page was written for PHP 7.0 which is but outdated right now and current versions of Nextcloud require at least PHP 7.2. We noticed that right in the middle of the task ans then we had to reinstall all PHP packages again. And while being not that difficult it was kind of an effort to retype all those install commands.

  • It was hard to figure out the correct configuration for the LDAP app to make the configuration work with the HdM MI LDAP sever in the "Advanced" and "Expert" tabs.
