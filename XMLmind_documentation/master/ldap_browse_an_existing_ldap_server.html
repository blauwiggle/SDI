1. Browse an existing LDAP Server

We browsed the LDAP Server at ldap1.mi.hdm-stuttgart.de using Apache Directory Studio. First we connected as anonymous user. Finding our names was easy by just typing in uid and our respective HdM token in the filter fields. Then we reconnected with our user DNs. The difference was that we were now able to see our enrolment numbers.

Using the ldapsearch tool wasn't too difficult. We just had to install the ldap-utils package. Then we reconfigured the ldap.conf file under /etc/ldap with our server parameters. Then we could just execute for example the following command:

ldapsearch -x -H ldap://ldap1.mi.hdm-stuttgart.de -b "dc=hdm-stuttgart,dc=de" -s sub "uid=fs120"
