3. Mounting our share from a remote client

We could remotely mount our share with the following command:

sudo mount -t cifs //sdi8b.mi.hdm-stuttgart.de/root/sambashare /mnt/test -ouser=root

This command responded with a password prompt in which we had to insert the samba password.

We could then check the content of the mounted share:

ls /mnt/test
test.txt

cat /mnt/test/test.txt
hallo test samba%
